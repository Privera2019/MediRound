MediRound Patient Monitoring System

A React + Vite + Tailwind + Firebase dashboard used by hospital staff, managers, and admins to view patient details, check-in history, rounding status, user management, and downloadable reports.
Requirements

You must have the following installed:

Tool	Version
Node.js	20.x or newer
npm	Included with Node
Git	Optional

Installation

Download or clone the project

If downloaded as a ZIP:
Extract the zip somewhere on your computer.

If using Git:

git clone https://github.com/your-repo/mediround-dashboard.git
cd mediround-dashboard

Install Dependencies

Open a terminal inside the project folder and run:

npm install

After Run the Development Server
npm run dev
